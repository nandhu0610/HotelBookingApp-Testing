# HotelBookingApp-Testing
Programming language - JAVA (with Selenium)
Design Pattern - Page Object Model
Framework - TestNG
Approach - Data Driven approach using DataProvider
Build - Maven

BaseTest --> src/main/java/base
Pages --> src/main/java/pages
Testcase --> src/test/java/testcases
Utilities --> src/test/java/utilities

Logs - log4j --> src/test/resources/logs
Report - Extent report --> src/test/resources/reports
Test data - MS Excel using apachePOI --> src/test/resources/testdata
Config files - properties file --> src/test/resources/configfiles
Runner file - testng.xml -->testrunner
